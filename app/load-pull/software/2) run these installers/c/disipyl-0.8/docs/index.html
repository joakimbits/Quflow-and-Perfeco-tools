This document provides information about disipyl, an object-oriented Python library for generating scientific plots.

Documentation

The available documentation on disipyl is of two types:

  1. A set of hand-crafted documents which provide an overview, tutorial, and exposition on how and why disipyl is put together the way that it is.
  2. Documentation automatically generated from source files (using pydoc).

Additionally, you should probably get a hold of the documentation that comes with DISLIN.
